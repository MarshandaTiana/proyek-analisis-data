# E-Commerce Dataset
## Setup Environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel
```
## Run streamlit dashboard
```
streamlit run dashboard.py
```
